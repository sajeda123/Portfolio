Automatic Face Recognition

With the use of Raspberry Pi kit,we aim at making the system cost effective and easy to use,with high performance.

IOT Based Fish Health Monitoring System

This is Raspberry Pi based system .In this system I used Temperature sensor,Humidity Sensor,Ph Sensor that is used for collecting water and shrimp health related data to the local database for processing and analysis. At the same time, it will send notification in user's Android phone based on pond's condition.

Spin Coater Machine

This is research based system and developed by Arduino Uno. It is used to make semiconductor thin films with a centrifugation force. A brushless direct current motor was selected as an actuator, its mathematical model is presented including the mechanical and electrical dynamics. This system code is written by Arduino Editor. This system was developed in the BCSIR(Bangladesh Council of scientific & Industrial Research)lab. I was worked in the coding section.

IOT Based Weather Notice System

IoT based weather notice system is developed by Raspberry Pi.The project is embedded with temperature, humidity, and rain sensors that can monitor the weather conditions and provide live data to the local database and displays the data on the LCD screen.Everyone can directly check the weather through online without having to rely on the reports of weather forecasting agencies.
